# <a name="device-relay"></a>장치 릴레이

디바이스 릴레이는 로컬 디바이스에서 원격으로 작업을 명령하거나 시작하는 데 사용됩니다.

작업에는 원격 앱 시작, 원격 메시징, 연결할 원격 시스템을 검색하고 파악하는 프로세스가 포함될 수 있습니다.

클라이언트 디바이스에서 디바이스 릴레이 시나리오를 시작하려면 왼쪽 탐색 창에서 원하는 개발 플랫폼을 시작하세요.
